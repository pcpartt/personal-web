BRIEFING HERO

ANUNCIANTE: Hero España S.A.

MARCA: HERO.

PRODUCTO/SERVICIO: Envase para mermelada de un sabor a elegir libremente.

TARGET: Personas que consumen mermeladas y conservas de frutas.

OBJETIVO: Diseñar un packaging tractivo para el nuevo producto. La etiqueta debe contener el logo de Hero, los contenidos obligatorios informativos y un uso alternativo para el tarro.

PIEZAS:

- Etiqueta de presetnación.

- Pegatina para el tapón.

PRESENTACIÓN DEL PROYECTO

El diseño del packaging cuenta con una etiqueta envolvente para el tarro con toda la información necesaria alimentaria y con un uso alternativo culinario. Para la tapa se ha diseñado una pegatina redondeada. Además al destapar el bote, en la parte interior de la tapa el consumidor podrá ver un segundo uso divertido y ecológico para reutilizar el tarro.
